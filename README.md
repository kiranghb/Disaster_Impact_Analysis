-> ## Disaster Impact Analysis ## <-

-> Project Overview

The **Disaster Impact Analysis** project performs **Exploratory Data Analysis (EDA)** on a disaster dataset to understand the impact of natural disasters on affected regions. The analysis identifies trends, relationships, outliers, and statistical patterns using Python data analysis libraries.

This project demonstrates the complete EDA workflow, including data cleaning, visualization, correlation analysis, multicollinearity detection, and hypothesis testing.


-> Objectives

- Analyze disaster-related data.
- Identify missing values and outliers.
- Understand the distribution of variables.
- Discover relationships between different features.
- Detect multicollinearity using VIF.
- Perform statistical hypothesis testing.
- Visualize important insights using charts and graphs.


-> Dataset

**Dataset Name:** `disaster_impact_dataset.csv`

The dataset contains disaster-related information such as:

- Event Type
- Region
- Population Affected
- Infrastructure Damage
- Economic Loss
- Fatalities
- Relief Funds
- Response Time
- Warning Issued
- Building Density
- Total Damage

-> How to Run

1. Clone the repository

```bash
git clone https://github.com/kiranghb/Disaster_Impact_Analysis.git
```

2. Navigate to the project folder

```bash
cd Disaster_Impact_Analysis
```

3. Install the required libraries

```bash
pip install pandas numpy matplotlib seaborn scipy statsmodels
```

4. Open the notebook

```bash
jupyter notebook Disaster_Impact_Analysis.ipynb
```
